Просмотр разделов

Для просмотра разделов лекций, лабораторных работ, ОКР, а также для прохождения тестов, необходимо выбрать соответствующий раздел, нажав на соответствующую кнопку на боковой панели, расположенной справа от основного контента. Пользователь имеет право просматривать и скачивать материалы. Если некоторые лекции отсутствуют в списке, скорее всего, они скрыты, и в таком случае, необходимо обратиться к преподавателю для выдачи доступа к конкретным записям или разделам, а также к тестам.

Для просмотра дополнительной информации о записи, требуется развернуть элемент, нажав на него. Внизу элемента появится его описание и файлы, которые можно скачать на компьютер.

В разделе "Тесты" пользователь может проходить тесты, нажав на кнопку "Начать". В тестах предоставлены варианты ответов, из которых нужно выбрать один правильный. После завершения теста и нажатия кнопки "Отправить", сервер отправит результаты пользователю. Если пользователь в режиме администратора, то результаты теста не будут записаны в базу данных, чтобы не засорять ее ненужными данными.

Помощь администратору

В режиме администратора появляются дополнительные функции: изменение описания и добавление/удаление файлов в любом разделе на любой записи, кроме тестов.

Чтобы изменить описание записи в разделе, нужно развернуть запись и нажать на значок редактирования около описания. Затем появится поле ввода нового описания. Если не хотите вводить новое описание, оставьте поле пустым.

Чтобы прикрепить файл, нужно нажать кнопку "Загрузить". После выбора файла он отобразится на записи. Если нужно удалить файл, нажмите крестик рядом с названием файла.

К дополнению администратору добавляется панель "Проверки знаний", которая позволяет полноценно изменять все записи. После выбора одного из разделов появляются все записи, включая скрытые. По нажатию кнопки "Редактировать" появляются поля ввода новых данных для определенной записи. Новые данные будут изменены только после нажатия кнопки "Принять". Кнопка "Добавить" создает новую запись со значениями по-умолчанию. Кнопка "Удалить" удаляет запись из раздела и базы данных.

Для смены доступа к определенным тестам и записям разделов нужно в режиме редактирования записи изменить значение доступа на "Да" или "Нет".

Для отображения результатов тестов есть вкладка "Статистика", где указывается ФИО проходящего тест, его группа, имя теста и итоговая оценка.

Для импорта данных в приложение нужно подготовить файл Excel и отправить его на сервер. Для правильного импорта данных следует учитывать следующее:

  • Сервер считывает данные из первой страницы файла Excel и только со второй строки листа, оставляя место для наименования столбцов.
  • Сервер считывает данные только первых четырех столбцов и только из определенных разделов: "Лекции", "лабораторные работы", "ОКР".
  • Столбцы должны быть строго в порядке и иметь определенные данные. При этом учитывается регистр.
  • Первый столбец - наименования (обязательное поле).
  • Второй столбец - описание (необязательное поле).
  • Третий столбец - доступ, может принимать значения только "Да" и "Нет". По умолчанию будет использоваться значение "Нет", если поле пустое (необязательное поле).
  • Четвертый столбец - раздел, может иметь только разделы "Лекции", "лабораторные работы", "ОКР" (обязательное поле).
